## Recommended: Coca-Cola Bottle Images

Upload 3 vertical bottle images in different formats:

- `cocacola.jpg` - JPEG format
- `cocacola.png` - PNG format  
- `cocacola.gif` - GIF format

## Why Vertical Images?

This module was created to solve the problem of displaying vertical
product images (like bottles) uniformly in product grids.

## Test Image Specs

**Recommended dimensions:**
- Width: 400-600px
- Height: 1200-1800px (vertical orientation)

**Formats:** JPG, PNG, GIF

## How to Test

1. Upload images to a ProcessWire page
2. Add to template: `<?php include('./test-squareimages.php'); ?>`
3. View page in browser
4. See 50+ automated tests run!
